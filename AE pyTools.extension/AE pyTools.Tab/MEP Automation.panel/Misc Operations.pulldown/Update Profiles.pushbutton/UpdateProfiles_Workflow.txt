Run Update Profiles after adding tags, keynotes, or text notes to equipment that
already carries Element_Linker metadata, or after changing parameter values on
those elements. Select the target equipment first. The tool scans only the
selection, finds their LED definitions,
and merges the new annotations and parameter values into the active YAML.
Proximity capture for nearby keynotes/text notes is currently disabled.

Once saved, rerun Place Linked Elements or Place Annotations so new views pick
up the updated definitions.
